# warehouse_stock_analysis

This project analyzes warehouse inventory data to better understand stock distribution, pricing patterns, and brand performance. The goal of the analysis is to identify key drivers of inventory value and explore relationships between product characteristics and stock levels.

## Project Overview

The analysis includes:

- Exploratory Data Analysis (EDA)
- Descriptive statistics
- Correlation analysis
- Statistical association tests
- Predictive modeling using Random Forest

## Key Insights

- Inventory value is concentrated in a relatively small number of products.
- Certain brands dominate warehouse stock value and distribution.
- Pricing variables have a strong influence on total inventory value.
- Machine learning models can help identify the most important predictors of inventory value.
